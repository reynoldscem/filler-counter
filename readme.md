# Filler Counter

Just `pip install` this repo, and use as follows:

```
filler-counter naruto bleach naruto-shippuden
```

Help available via:

```
filler-counter -h
```

Example output:

```
naruto:
    Filler count:    94
    Canon count:    126
    Total count:    220
----------------------------------------

Could not find 'not-real-one'!
----------------------------------------

```
